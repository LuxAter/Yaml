    Yaml Class¶

    Defined in header <yaml/yaml.hpp>

    class Yaml;

    yaml::Yaml is a structured class, that mimics yaml file structure. This class can either be a single type value (bool, float, int, string), or it can be an ordered list of values (std::vector), or it can be a sorted list of associated pairs (std::map).
